🚀 Landing Page de Hospedagem

Este projeto é uma landing page moderna para serviços de hospedagem, desenvolvida com foco em conversão, performance e experiência do usuário.

Toda a aplicação foi construída 100% por mim, sem uso de IA, desde o design até a implementação, reforçando minhas habilidades práticas em desenvolvimento front-end.

📌 Sobre o Projeto

A landing page tem como objetivo apresentar serviços de hospedagem de forma clara, atrativa e persuasiva, guiando o usuário até a tomada de decisão (contratação/contato).

O layout foi pensado estrategicamente para destacar benefícios, gerar confiança e facilitar a navegação.
